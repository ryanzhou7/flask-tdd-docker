# Readme

## Run w/o docker

- `$ export FLASK_APP=src/__init__.py`
- `$ export FLASK_DEBUG=1`
- `$ python manage.py run`

## Run with docker

- https://testdriven.io/blog/docker-best-practices/
- `$ docker-compose build`
- `$ docker-compose up -d`
- `$ docker-compose up -d --build # update container`
